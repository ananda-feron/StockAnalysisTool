# Hawk Fund Stock Analyzer
An easy-to-use site that analyzes stock. It includes visuals, calculations, vocabulary, insights, and a report generator. 

As a former Student Stock Analyst for the SUNY New Paltz Hawk Fund, I was inspired by the stock analysis reports I created to assist the club's investment decisions that were made on behalf of the school. Each report required:

• Selecting a stock
<br>• Conducting quantitative analysis
<br>• Conducting fundamental analysis
<br>• Conducting technical analysis
<br>• Synthesizing findings
<br>• Making a buy/hold/sell recommendation

My goal in making this site is to help current students in the club more easily navigate and produce Hawk Reports.
